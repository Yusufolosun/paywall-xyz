Installation

CDN (Recommended)

<script src="https://cdn.paywall.xyz/widget.js"></script>

NPM

npm install @paywall/widget

Quick Start

Add the widget script and mark content with data attributes:

<script src="https://cdn.paywall.xyz/widget.js"></script>

<div data-paywall-id="1" data-paywall-price="0.5">
  <h2>Premium Article</h2>
  <p>This content is locked until payment.</p>
</div>

<script>
  new PaywallWidget({
    contractAddress: 'ST1PQHQKV0RJXZFY1DGX8MNSNYVE3VGZJSRTPGZGM.paywall-v1',
    network: 'testnet'
  }).init();
</script>

Configuration

Data Attributes

  • data-paywall-id - Unique content identifier (integer)
  • data-paywall-price - Price in STX (decimal)

Widget Options

interface PaywallConfig {
  contractAddress: string;  // Deployed contract address
  network: 'testnet' | 'mainnet';
}

API Reference

PaywallWidget Class

const widget = new PaywallWidget(config);
await widget.init();

Methods

  • init(): Promise<void> - Initialize widget and scan DOM

Smart Contract

Testnet

ST1PQHQKV0RJXZFY1DGX8MNSNYVE3VGZJSRTPGZGM.paywall-v1

Functions

  • register-content - Register content with pricing
  • unlock-content - Pay to unlock content
  • has-access - Check if user has access

Revenue Split

90% to content creator, 10% platform fee

Styling

CSS Classes

  • .paywall-locked - Applied when content is locked
  • .paywall-unlocked - Applied after successful unlock
  • .paywall-unlock-btn - Unlock button element
